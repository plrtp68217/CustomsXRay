### Описание проекта

**«Таможня: Рентген-Контроль» (Customs: X-Ray)** — мультиплеерная (1v1) веб-игра на тему таможни.

- **Жанр:** Аркада / Стратегия на внимательность с элементами обмана.
- **Игроки:** 2 человека (Контрабандист vs Таможенник).
- **Стек:** HTML/CSS/JS (SPA) + Node.js + Express + Socket.IO.
- **Стиль:** Flat / hi-tech, тёмный фон #0B1124, бирюзовые акценты #00F0FF, оранжевые опасные зоны #FF6B00.

#### Игровой цикл
1. **Лобби:** хост создаёт комнату с кодом, второй подключается по коду. Старт — только у хоста.
2. **Раунд (3 всего, роли меняются):**
   - **Фаза 1 — Укладка (30с):** контрабандист Drag-and-Drop'ом совмещает силуэт контрабанды с пустой полостью внутри легального предмета. Индикатор совпадения 0–100%, порог идеала ≥85%.
   - **Фаза 2 — Рентген (30с):** таможенник сканером проверяет 3–5 предметов на ленте. Хорошо спрятанная контрабанда почти не видна, плохо — красный силуэт + вибрация. Клик → «Изъять».
3. **Очки:** контрабандист +1, если контрабанду не нашли; таможенник +1 за верное изъятие; таможенник −1 за ложное изъятие.
4. **Результаты:** после 3 раундов — экран статистики и победитель.

#### Архитектура файлов
- `package.json` — зависимости и скрипты запуска.
- `server.js` — Express + Socket.IO, игровая логика, валидация, авторитарные таймеры.
- `public/index.html` — SPA-разметка всех экранов.
- `public/css/style.css` — стили (flat/hi-tech, адаптив).
- `public/js/geometry.js` — утилиты геометрии (point-in-polygon, площадь, совпадение силуэтов). Shared (Node + browser).
- `public/js/items.js` — определения предметов (легальный силуэт + полость + контрабанда). Shared.
- `public/js/audio.js` — Web Audio API звуки без внешних файлов.
- `public/js/game.js` — клиентская логика, рендер на Canvas, Socket.IO-клиент.

#### Запуск
```bash
npm install
npm start
```
Сервер на `http://localhost:3000`.

## Редизайн под новый сценарий «Досмотр машины»

- **[2026-06-22]** Полная переделка игрового цикла под новый сценарий:
  - **Машина** с 6 слотами груза (2 ряда × 3 колонки). row 0 = передний (перекрывает задний на рентгене), row 1 = задний (перекрыт).
  - **Фаза укладки (контрабандист, PACK_TIME_MS):** 6 слотов заполнены легальным товаром (с весами). Контрабандист заменяет один слот на контрабанду (drag силуэта в слот + ручка поворота). Умение: выбрать слот с близким весом (обмануть весы) и задний ряд (обмануть рентген).
  - **Фаза досмотра (таможенник, INSPECT_TIME_MS), инструменты:**
    1. **Рентген** (лимит 3) — размытые силуэты (canvas blur), задний ряд перекрыт передним и больше размыт.
    2. **Покачать** (лимит 3) — перетасовка позиций груза (posIndex/row/col/x/y), может открыть ранее перекрытое.
    3. **Взвесить** — фактический vs задекларированный вес; Δ = контрабанда − заменённый товар.
    4. **Допрос** (лимит 3) — клик слота → вопрос контрабандисту (второй игрок отвечает в реал-тайме через overlay); сервер добавляет «стресс-tell» (😰/😌) на слоте с контрабандой (вероятностно).
  - Машину напрямую не вскрывать — только **Изъять слот** или **Пропустить** (одно действие завершает раунд).
  - **Счёт:** изъял контрабанду → таможенник +1; ошибся/пропустил → контрабандист +1; ложное изъятие → таможенник −1.
  - Изъятие проверяется по **текущей позиции** контрабанды (posIndex), а не по исходному слоту — после покачивания контрабанда может быть на другом месте.
